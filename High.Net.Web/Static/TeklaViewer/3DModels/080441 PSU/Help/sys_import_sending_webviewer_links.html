Sending WebViewer links

There are two ways to send links from Tekla WebViewer:

  • Send WebViewer link
  • Send URL link

Send WebViewer link

With this tool you can send a link to one view at a time. The receiver copies the text string and pastes it on the WebViewer model and the view name appears on the Named views list.

When you want to send many views to other parties, copy the text strings pointing to the views into a text file and send the file to receivers.


See also Receiving WebViewer models.

Send URL link

With this tool you can send an URL link to the model to recipients who have access to the folder where your published model is located.
