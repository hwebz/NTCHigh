Extract all of the files from the winzip file first. 

Double click on the file "index.html" this will load the web model in your browser.  

Depending on you security settings you may get a bar that says " To help protect your security, 
Internet Explorer has restricted this webpage from running scripts or 
Active X controls that could access your computer. Click here for options..."

If you get this bar, click the bar and select "Allow blocked content"
